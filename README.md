# P-NATS DataBase
P-NATS DataBase is a collection of nucleic acid structures purified by [P-NATS](https://github.com/S-Ando-Biophysics/P-NATS) for use in molecular replacement without preprocessing. 

You can access this database via the following URL. https://s-ando-biophysics.github.io/P-NATS-DB/
